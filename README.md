1. Clone the repo

2. run "cd student-management"

3. create a file ".env"

4. copy contents of .env.example to .env

5. configure your database credentials in .env

6. run "composer install"

7. run "php artisan key:generate"

6. run "php artisan migrate --seed"

5. run the project
